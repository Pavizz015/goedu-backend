Почему Go выбирают

  • Простой и минималистичный синтаксис
  • Очень быстрая компиляция в один бинарник
  • Встроенная многопоточность через горутины
  • Сборщик мусора без накладных расходов
  • Большая стандартная библиотека

Go стал фундаментом современной cloud-инфраструктуры...

Docker Kubernetes Terraform CockroachDB Caddy Hugo etcd InfluxDB

Установка Go

Go устанавливается на Windows, macOS и Linux...

  1. 1Зайди на go.dev/dl/ и скачай .msi-инсталлятор.
  2. 2Запусти установщик. Установи Go в стандартный каталог.
  3. 3Открой PowerShell или CMD и проверь установку:
go version

Если видишь "go version go1.22 ..." — Go установлен правильно.

  1. 1Самый простой способ — через Homebrew:
brew install go
  1. 2Альтернатива — скачать .pkg с go.dev/dl/ и запустить.
  2. 3Проверка установки:
go version

Если видишь "go version go1.22 ..." — Go установлен правильно.

  1. 1Скачай tar.gz архив с go.dev/dl/ и распакуй в /usr/local:
sudo rm -rf /usr/local/go
sudo tar -C /usr/local -xzf go1.22.linux-amd64.tar.gz
  1. 2Добавь Go в PATH через ~/.bashrc или ~/.zshrc:
export PATH=$PATH:/usr/local/go/bin
  1. 3Перезагрузи терминал и проверь:
source ~/.bashrc
go version

Если видишь "go version go1.22 ..." — Go установлен правильно.

Настройка VS Code

VS Code — самый популярный редактор для Go...

  1. 1Скачай VS Code с code.visualstudio.com и установи.
  2. 2Открой VS Code, нажми Ctrl+Shift+X — откроется панель расширений.
  3. 3Найди расширение Go от Go Team at Google и установи.
  4. 4Открой любой .go файл — VS Code предложит установить gopls, dlv и другие. Нажми Install All.
  5. 5Готово — автодополнение, переходы, отладчик и автоформатирование при сохранении.

Настройка GoLand

GoLand — IDE от JetBrains специально для Go...

  1. 1Скачай GoLand с jetbrains.com/go. Студенты — через JetBrains Student Pack.
  2. 2Запусти установщик, прими лицензию.
  3. 3При первом запуске GoLand сам найдёт Go и подхватит GOPATH/GOROOT.
  4. 4Создай новый проект: File → New → Project → Go.
  5. 5Всё готово — GoLand сам подскажет когда нужны модули и отформатирует код.

Первая программа

Создай файл main.go и вставь этот код:

package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}

Запусти из терминала:

go run main.go

Должно вывести: Hello, World!

⌨

Основные команды Go

go run main.goЗапустить программу без сборки бинарника
go buildСкомпилировать в исполняемый файл
go mod init projectСоздать модуль (новый проект)
go mod tidyПодтянуть/убрать зависимости в go.mod
go get github.com/...Установить стороннюю библиотеку
gofmt -w .Отформатировать код по стандарту Go
go test ./...Запустить тесты
go vet ./...Проверить код на подозрительные конструкции
